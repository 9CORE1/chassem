# 🚀 차은정 포트폴리오 웹 애플리케이션 주요 기능 요약

이 프로젝트는 **취업진로 컨설팅, 영상콘텐츠 제작, 그리고 IT 미래기술** 세 가지 분야의 전문성을 담은 **차은정님의 융합형 포트폴리오 웹 애플리케이션**입니다.

## 🌟 핵심 기능 (Core Features)

### 1. 포트폴리오 동적 필터링 및 상세 모달
- **카테고리 필터링:** 취업진로(Career), 영상콘텐츠(Media), IT 미래기술(IT)의 3대 카테고리별로 포트폴리오 프로젝트를 동적으로 필터링하여 탐색할 수 있습니다.
- **상세 보기 모달:** 프로젝트를 클릭하면 상세 역할, 성과 하이라이트, 사용 기술 스택이 팝업으로 나타나며, 관련 유튜브 영상 임베드(재생 구간 설정 지원) 및 이미지를 제공합니다.

### 2. 성장 스토리 타임라인 시각화
- **이력 분류:** 경력사항, 학력사항, 자격/수료, 저작활동 등 작성자의 이력을 시간 순서대로 분류하여 보기 쉬운 타임라인 형태로 시각화하여 렌더링합니다.

### 3. 실시간 콘텐츠 관리를 위한 관리자 모드 (Admin Mode)
- **이메일 인증 활성화:** 화면 상의 잠금(Lock) 아이콘을 클릭하고 이메일 인증을 통해 관리자 권한을 획득하면 실시간으로 사이트 상에서 콘텐츠를 수정 및 추가, 삭제할 수 있는 편집 모드로 전환됩니다.
- **임시 저장:** 편집 중인 내용은 브라우저의 `localStorage`에 자동 보관되어 세션이 만료되거나 창이 닫혀도 작성 내용을 안전하게 보존합니다.

### 4. 하이브리드 데이터 저장 시스템
- **로컬 서버 저장:** 로컬 개발 및 실행 환경에서는 Node.js Express 백엔드의 `/api/save` API를 통해 `data.json` 파일에 즉시 저장됩니다.
- **GitHub API 연동 저장 (정적 호스팅용):** GitHub Pages와 같이 백엔드 서버가 없는 정적 호스팅 환경에서도 GitHub Personal Access Token(PAT)을 활용하여 GitHub 리포지토리에 직접 데이터 및 업로드한 이미지를 커밋하고 푸시하여 원격 동기화할 수 있습니다.

### 5. 자동 이미지 디코딩 및 업로드
- 관리자 모드에서 첨부된 Base64 인코딩 형식의 이미지를 서버 또는 GitHub API를 통해 `images/project-[id].jpg` 파일로 자동 디코딩하여 저장하고, 데이터 파일(`data.json`) 내 경로를 상대 경로로 치환해 줍니다.
