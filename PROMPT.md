# 🚀 융합형 포트폴리오 웹 애플리케이션 구축 요청

이 프로젝트는 취업진로 컨설팅, 영상콘텐츠 제작, IT 미래기술 3가지 분야의 융합형 스페셜리스트 '차은정'님의 웹 포트폴리오 애플리케이션입니다. 아래 명세에 따라 로컬 구동과 정적 호스팅(GitHub Pages 등)이 모두 가능한 하이브리드 웹 애플리케이션을 개발해 주세요.

## 🛠️ 1. 기술 스택 및 구조
- **프론트엔드:** HTML5, Vanilla CSS(반응형, Glassmorphism UI, 기본 다크모드/라이트모드 테마 지원), Vanilla JavaScript (순수 DOM 제어)
- **백엔드 (Local Run Mode):** Node.js & Express (로컬 데이터/이미지 저장용)
- **데이터 저장 모델:** 단일 JSON 파일(`data.json`) 기반의 데이터 관리

## 🌟 2. 핵심 기능 요구사항
1. **포트폴리오 카테고리 필터링 & 상세 모달:**
   - Career(취업진로), Media(영상콘텐츠), IT(IT 미래기술)의 3가지 카테고리로 필터링 제공.
   - 프로젝트 클릭 시 상세 정보 모달 오픈 (기술 스택, 성과 하이라이트, 유튜브 임베드 재생구간 설정 기능 포함).
2. **성장 스토리 타임라인:**
   - 학력, 경력, 자격, 수료 이력을 시간 순으로 분류 및 동적 렌더링.
3. **양방향 동기화 관리자 모드 (Admin Mode):**
   - 이메일 검증 기반 잠금 해제(Lock 아이콘 클릭 시 편집 활성화).
   - 화면상에서 콘텐츠 실시간 추가/수정/삭제 가능 및 `localStorage` 임시 보관 지원.
4. **하이브리드 데이터 저장:**
   - 로컬 구동 시 `/api/save` API를 통해 로컬 `data.json`에 직접 저장.
   - 정적 호스팅 구동 시 GitHub API(Personal Access Token 연동)를 통해 원격 리포지토리에 직접 `data.json`과 이미지를 커밋 및 푸시하여 동기화.
5. **자동 이미지 변환:**
   - 업로드된 Base64 인코딩 이미지를 서버 또는 GitHub API를 통해 `images/project-[id].jpg` 파일로 물리적 저장하고 경로 자동 치환.

## 📂 3. 프로젝트 초기 파일 구조
프로젝트 루트 폴더에 아래의 구조로 파일들을 초기화하고 구현해 주세요.
- `package.json` (Express, CORS 라이브러리 추가)
- `server.js` (Express 로컬 서버 소스)
- `index.html` (포트폴리오 UI 마크업)
- `styles.css` (글래스모피즘 및 애니메이션 포함 스타일시트)
- `app.js` (포트폴리오 렌더링, 관리자 로직 및 동기화 스크립트)
- `data.json` (포트폴리오, 타임라인 정보가 들어갈 기초 데이터 뼈대)

먼저, 프로젝트 구성을 위한 `package.json`과 로컬 서버 `server.js`, 그리고 기초 데이터 포맷인 `data.json`을 작성하는 것부터 순차적으로 시작해 주세요.
